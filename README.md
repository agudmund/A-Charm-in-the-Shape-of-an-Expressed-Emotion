# A Charm in the Shape of an Expressed Emotion

> *The last of the charm in the shape of an expressed emotion slipped quietly off the screen, certain it belonged on a wall in a room it had never seen, For Enjoying*  
> *Built using a single shared braincell by Yours Truly and various Intelligences*

The print-and-frame shelf: where pieces leave the screen and go up on the living-room walls.

Whatever is chosen to become physical gathers here (art, stills, the occasional moving piece) until it is sent to the printer and framed in the real world. The name is the spirit of it: each one a small charm, an emotion given a shape you can hang. The namesake piece, "A charm in the shape of an expressed emotion", is the heart of the collection.

This is a curated shelf, not a working directory. If something lives here, it has earned a candidacy for the wall: nothing throwaway, everything chosen.
